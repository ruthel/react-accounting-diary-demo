# react-accounting-diary-demo

Demo app for [react-accounting-diary](https://www.npmjs.com/package/react-accounting-diary).

## Setup

```bash
pnpm install
pnpm dev
```

## Features Demonstrated

- Callback props (`onChange`, `onAdd`, `onEdit`, `onDelete`)
- Pagination (`pageSize={10}`)
- Grand total & balance validation
- Column sorting
- Keyboard navigation
- Empty state with CTA

## Stack

- React 19
- Vite 8
- TypeScript
