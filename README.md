# react-accounting-diary-demo

Demo app for [react-accounting-diary](https://www.npmjs.com/package/react-accounting-diary).

> **[Live Demo →](https://react-accounting-diary-demo.vercel.app)**

## Setup

```bash
pnpm install
pnpm dev
```

## Features Demonstrated

- Callback props (`onChange`, `onAdd`, `onEdit`, `onDelete`)
- Validation callbacks (`onBeforeAdd`, `onBeforeEdit`, `onBeforeDelete`)
- Controlled mode with external state
- Headless hook (`useAccountingDiary`)
- Imperative ref API
- Ledger view toggle
- Category & tags
- JSON import/export
- Pagination (`pageSize={10}`)
- Grand total & balance validation
- Sortable columns (date, account, amount)
- Search & date range filter
- Dark/Light theme
- Keyboard navigation & ARIA
- Empty state with CTA
- i18n / Localisation
